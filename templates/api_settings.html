🤖 AI 협업 기능을 사용하려면 API 키가 필요합니다

GPT, Claude, Gemini와 협업하여 로또 번호를 생성하려면 각 AI 서비스의 API 키를 설정해야 합니다. 모든 키를 설정할 필요는 없으며, 설정된 AI만 협업에 참여합니다.

1🤖 OpenAI (GPT-4)

OpenAI API 키는 OpenAI Platform에서 발급받을 수 있습니다.

2🧠 Anthropic (Claude)

Anthropic API 키는 Anthropic Console에서 발급받을 수 있습니다.

3💎 Google (Gemini)

Google API 키는 Google AI Studio에서 발급받을 수 있습니다.
🔒 보안 정보
  • API 키는 암호화되어 안전하게 저장됩니다.
  • API 키는 로또 번호 생성 용도로만 사용됩니다.
  • 언제든지 설정을 변경하거나 삭제할 수 있습니다.
  • AI 협업 기능을 사용하지 않으면 기본 분석 방식이 사용됩니다.
